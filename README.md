# LiveCuration

ライブ配信・ライブコマース・SNSの最新情報キュレーションアプリ

## Features
- TikTok, Instagram, Pococha, REALITY, SHOWROOM等の最新ニュースを一覧表示
- プラットフォーム別タブフィルタリング
- キーワード検索
- お気に入り機能
- レスポンシブデザイン（モバイル〜デスクトップ対応）
- RSS自動取得（PR TIMES, note）

## Tech Stack
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- lucide-react
- rss-parser


Deployed with Vercel
